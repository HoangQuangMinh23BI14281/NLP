# Information Extraction from Job Descriptions Using Natural Language Processing

## Project Overview
This project applies Natural Language Processing (NLP) techniques to automatically extract key entities from unstructured Job Descriptions (JDs). 

**Core Entities Extracted:**
* **Skills:** Technical and soft skills required for the position.
* **Salary:** Compensation details (ranges, specific amounts, or hourly rates).
* **Location:** Work location (City, Country, Remote/Hybrid status).

## Models
To achieve high accuracy in Named Entity Recognition (NER) and information extraction, we experimented with the following Transformer-based models:
